tm1-ui-element-list-item
directive in module tm1Ui

Description

Used for indenting the elements generated by tm1-ui-element-list, and providing standard Canvas icons for leaf and consolidated elements. Ensure that tm1-calculate-top-level attribute is set to true for the tm1-ui-element-list where this will be used.

Usage

as element:
<tm1-ui-element-list-item
       tm1-item="{object}"
       [tm1-item-display="{string}"]
       [tm1-indent="{boolean}"]
       [tm1-ui-class="{string}"]>
</tm1-ui-element-list-item>

Directive info

  • This directive creates new scope.
