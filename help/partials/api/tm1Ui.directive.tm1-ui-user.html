tm1-ui-user
directive in module tm1Ui

Description

For retrieving user information from the instances.

Usage

as element:
<tm1-ui-user
       [tm1-instance="{string}"]
       [tm1-ui-class="{string}"]
       [ng-model="{object}"]>
</tm1-ui-user>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
tm1-instance
(optional)
string

Instance name for this directive to connect to. WATCHED

tm1-ui-class
(optional)
string

Add other custom classes into this directive.WATCHED

(default: "tm1-ui-user")

ng-model
(optional)
object

Angular JSON object where the actual values are stored, for usage in other parts of the page.
