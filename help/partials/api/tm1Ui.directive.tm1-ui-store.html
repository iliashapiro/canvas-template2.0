tm1-ui-store
directive in module tm1Ui

Description

A directive that lets you store information and preferences in JSON formatted-string into TM1. Note that a special keyword, @UserName can be used to get the current active user in the session, and can be used in the elements.

Usage

as element:
<tm1-ui-store
       tm1-instance="{string}"
       tm1-cube="{string}"
       tm1-elements="{string}"
       tm1-save-delay="{=string}"
       ng-model="{object}">
</tm1-ui-store>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
tm1-instancestring

Instance name for this directive to connect to. REQUIRED

tm1-cubestring

TM1 Cube name where this will connect to. REQUIREDWATCHED

tm1-elementsstring

comma separated element names that identifies the cube's data point to get. Element names can be an ALIAS. REQUIREDWATCHED

tm1-save-delay=string

The number of milliseconds before the directive saves the value into TM1.

(default: "500")

ng-modelobject

Angular JSON object that will represent the information to be stored or retrieved into TM1 REQUIREDWATCHED

(default: object)
