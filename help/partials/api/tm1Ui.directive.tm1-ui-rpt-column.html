tm1-ui-rpt-column
directive in module tm1Ui

Description

Use this to define the column elements on your active form. This is directive is then used as part of tm1-ui-rpt-view.

Usage

as element:
<tm1-ui-rpt-column
       tm1-dimension="{string}"
       [tm1-element="{string}"]
       [tm1-subset="{string}"]
       [tm1-mdx="{string}"]>
</tm1-ui-rpt-column>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
tm1-dimensionstring

Dimension name to query. REQUIREDWATCHED

tm1-element
(optional)
string

If provided, this will be used in the columns. WATCHED

tm1-subset
(optional)
string

If provided, elements of this subset will be used in the columns. WATCHED

tm1-mdx
(optional)
string

If provided, this MDX will be used to retrieve the elements to use in the columns. WATCHED
