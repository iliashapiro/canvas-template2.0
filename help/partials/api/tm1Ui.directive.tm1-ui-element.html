tm1-ui-element
directive in module tm1Ui

Description

Used for populating a ng-model with an element's properties from a dimension. Returned data is an object of the element properties requested.

Usage

as element:
<tm1-ui-element
       tm1-instance="{string}"
       tm1-dimension="{string}"
       tm1-element="{string}"
       [tm1-attributes="{string}"]
       [ng-model="{object}"]>
</tm1-ui-element>

Directive info

  • This directive creates new scope.
