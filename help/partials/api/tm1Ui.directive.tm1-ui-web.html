tm1-ui-web
directive in module tm1Ui

Description

Allows you to display a cube view or an Excel workbook in TM1's Applications

Usage

as element:
<tm1-ui-web
       tm1-instance="{string}"
       [tm1-application="{string}"]
       [tm1-cube="{string}"]
       [tm1-cube-view="{string}"]
       [tm1-titles="{string}"]
       [tm1-show-tool-bar="{boolean}"]
       [tm1-access-type="{string}"]
       [tm1-width="{string}"]
       [tm1-height="{string}"]
       [tm1-ui-class="{string}"]>
</tm1-ui-web>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
