tm1-ui-chart-value
directive in module tm1Ui

Description

A child directive to always use in conjunction with tm1-ui-chart as a data source.

Usage

as element:
<tm1-ui-chart-value
       tm1-value="{string}"
       [tm1-label="{string}"]
       [tm1-label-empty="{boolean}"]
       [tm1-series="{string}"]
       [tm1-area="{boolean}"]
       [tm1-order-no="{string}"]
       [tm1-color="{string}"]
       [tm1-show-movement="{boolean}"]>
</tm1-ui-chart-value>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
tm1-valuestring

Custom value that can be fed into the graph. REQUIREDWATCHED if defined.

tm1-label
(optional)
string

Description of this data point as will be seen on the chartWATCHED if defined.

(default: "DNN")

tm1-label-empty
(optional)
boolean

To force label to be blank.

(default: "false")

tm1-series
(optional)
string

Data series where this data point will belong to.WATCHED

(default: "Data")

tm1-area
(optional)
boolean

To format a line chart as an area. Only need to be defined ONCE, and it will set the whole series as AREA, if used in a LINE chart.

(default: "false")

tm1-order-no
(optional)
string

The order or X-axis value of this data, if used in a LINE|WATERFALL chart.

(default: "NN")

tm1-color
(optional)
string

Assign a specific color to a data point in CSS color format (i.g. #FFXXXX or Red or rgb(1,1,1) ). Applicable to line charts only. For line series, the first color encountered on the DBR will be used on that whole series.

tm1-show-movement
(optional)
boolean

Used with WATERFALL chart. To indicate whether the bar rendered is movement only, or the full value.

(default: "true")
