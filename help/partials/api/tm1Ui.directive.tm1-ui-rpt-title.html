tm1-ui-rpt-title
directive in module tm1Ui

Description

Use this to define the title element on your active form. This is directive is then used as part of tm1-ui-rpt-view.

Usage

as element:
<tm1-ui-rpt-title
       tm1-dimension="{string}"
       [tm1-element="{string}"]>
</tm1-ui-rpt-title>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
tm1-dimensionstring

Dimension name to query. REQUIREDWATCHED

tm1-element
(optional)
string

Element name defining one of the titles on your active form. If not defined, the 'DefaultMember' will be used instead. WATCHED
