formatNumber
filter in module tm1Ui

Description

This filter allows you to format your numbers on your HTML page through the accounting.js library. See below for a simple sample usage:

<div> {{100 | formatNumber}} <!-- returns '100.00' --> </div>
<div> {{100 | formatNumber:5}} <!-- returns '100.00000' --> </div>

Usage

In HTML Template Binding

{{ formatNumber_expression | formatNumber:decimal }}

In JavaScript

$filter('formatNumber')(value, decimal)

Parameters

ParamTypeDetails
valuenumber

The value to be formatted.

decimalnumber

Number of decimal places to apply to the output number.

Returns

string

The formatted number.
