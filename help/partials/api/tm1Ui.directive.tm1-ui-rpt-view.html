tm1-ui-rpt-view
directive in module tm1Ui

Description

This is the parent directive that allows you to build an active form.

Directive Level APIs. Access via ng-model.

pageFirst()
For going to the first page.
pagePrevious()
For going to the previous page. This is one page prior to the current one.
pageNext()
For going to the next page. This is one page after the current one.
pageLast()
For going to the last page.
page()
For getting the current page.
pages()
For retrieving the total number of pages.
isPageSize(pageSize)
For checking if this is the current page size.

Usage

as element:
<tm1-ui-rpt-view
       tm1-instance="{string}"
       tm1-cube="{string}"
       [tm1-suppress-zero="{both|row|column|none}"]
       [tm1-page-size="{string}"]
       [tm1-show-message="{boolean}"]
       [ng-model="{object}"]>
</tm1-ui-rpt-view>

Directive info

  • This directive creates new scope.
