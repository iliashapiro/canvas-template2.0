tm1-ui-dbr-image
directive in module tm1Ui

Description

A DBR-like implementation for retrieving cube data containing paths to images uploaded via tm1-ui-upload from TM1

Usage

as element:
<tm1-ui-dbr-image
       tm1-instance="{string}"
       tm1-cube="{string}"
       tm1-elements="{string}"
       [tm1-refresh-group="{string}"]
       [tm1-sandbox="{string}"]
       [tm1-sandbox-class="{string}"]
       [tm1-ui-class="{string}"]
       [tm1-alt="{string}"]
       [ng-model="{object}"]>
</tm1-ui-dbr-image>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails

tm1-cubestring

string

An ID that will allow DBRs belonging to the same group to be refreshed only. Editing a DBR on this group refreshes only those having the same group as this.

string

Name of the sandbox that this DBR will be associated with.WATCHED
