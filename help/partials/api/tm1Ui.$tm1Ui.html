$tm1Ui
service in module tm1Ui

Description

This service allows you interact with your Canvas applications through your Controllers.

In the following example, a function was prepared to be able to call one of the services available under $tm1Ui, retrieving information about your Canvas application. This then gets displayed on your browser's console:


// this is a sample Angular controller created to support your HTML page
app.controller('SomeCtrl', ['$scope', '$rootScope', '$tm1Ui', function($scope, $rootScope, $tm1Ui) {

  // let us define here the function which will then utilize a Canvas service
  $scope.getCanvasInformation = function(){
	 $tm1Ui.applicationInfo().then(function(data){
  		console.info('Returned by this function - %o', data);
	 });
 };

 // this now calls the function  	
 $scope.getCanvasInformation();

}]);
