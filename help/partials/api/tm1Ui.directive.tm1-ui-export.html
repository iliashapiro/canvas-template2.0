tm1-ui-export
directive in module tm1Ui

Description

Export data of properly formatted HTML tables such as: 

<table>
 <thead>
 	<tr>
 		<th>header...</th>
 	</tr>
	</thead>
 <tbody>
 	<tr>
 		<td>data...</td>
 	</tr>
 </tbody>
 </table>

Usage

as element:
<tm1-ui-export
       tm1-target-id="{string}"
       [tm1-output-name="{string}"]
       [tm1-field-separator="{string}"]
       [tm1-field-qualifier="{string}"]
       [tm1-locale="{string}"]
       [tm1-ui-class="{string}"]>
</tm1-ui-export>

Directive info

  • This directive creates new scope.
