tm1-ui-session
directive in module tm1Ui

Description

This allows access to a pop-up UI where user will be able to logout

Usage

as element:
<tm1-ui-session
       [tm1-ui-class="{string}"]>
</tm1-ui-session>

Directive info

  • This directive creates new scope.
