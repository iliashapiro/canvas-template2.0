formatPercentage
filter in module tm1Ui

Description

This filter allows you to format your numbers to percentage on your HTML page through the accounting.js library. See below for a simple sample usage:

<div> {{0.3333333 | formatPercentage}} <!-- returns '33.33%' --> </div>
<div> {{0.3333333 | formatPercentage:5}} <!-- returns '33.33333%' --> </div>

Usage

In HTML Template Binding

{{ formatPercentage_expression | formatPercentage:decimal }}

In JavaScript

$filter('formatPercentage')(value, decimal)

Parameters

ParamTypeDetails
valuenumber

The value to be formatted as percentage.

decimalnumber

Number of decimal places to apply to the output number.

Returns

string

The formatted number in percentage.
