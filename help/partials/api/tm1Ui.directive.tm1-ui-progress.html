tm1-ui-progress
directive in module tm1Ui

Description

This directive will show a spinning gear icon whenever there is an pending request on the current page. You also place your own indicator or message by putting your content or image within the two special markups of this directive.

Usage

as element:
<tm1-ui-progress
       [tm1-ui-class="{string}"]>
</tm1-ui-progress>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
tm1-ui-class
(optional)
string

Add other custom classes into this directive.WATCHED

(default: "tm1-ui-progress")

Example

For the default activity indicator, using Font Awesome's spinning Cog icon 

<tm1-ui-progress></tm1-ui-progress>

To place your own custom content

<tm1-ui-progress><tm1-ui-progress-pending>Please Wait...</tm1-ui-progress-pending></tm1-ui-progress>

To place your own custom pending content AND content to show once it is ready

<tm1-ui-progress>
			<tm1-ui-progress-pending>Please Wait...</tm1-ui-progress-pending>
			<tm1-ui-progress-ready>Ready</tm1-ui-progress-ready>
		</tm1-ui-progress>
