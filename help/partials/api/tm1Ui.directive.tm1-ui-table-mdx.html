tm1-ui-table-mdx
directive in module tm1Ui

Description

For retrieving and displaying the result of a named MDX (defined in WEB-INF/resources/mdx_named.json file), in table format.

Usage

as element:
<tm1-ui-table-mdx
       tm1-instance="{string}"
       tm1-cube="{string}"
       tm1-mdx-id="{string}"
       [tm1-mdx-parameters="{object}"]
       [tm1-dimension-attribute="{object}"]
       [tm1-ui-class="{string}"]>
</tm1-ui-table-mdx>

Directive info

  • This directive creates new scope.
