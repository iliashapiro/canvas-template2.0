tm1-ui-element-list
directive in module tm1Ui

Description

Used for populating a ng-model with elements from a dimension. Returned data is an array, with key, level and alias|attributes properties.

Directive Level APIs

refreshData()
For refreshing the list of elements

Usage

as element:
<tm1-ui-element-list
       tm1-instance="{string}"
       tm1-dimension="{string}"
       [tm1-subset="{string}"]
       [tm1-attributes="{string}"]
       [tm1-mdx="{string}"]
       [tm1-calculate-top-level="{boolean}"]
       [tm1-api="{object}"]
       [tm1-options="{object}"]
       [ng-model="{object}"]>
</tm1-ui-element-list>

Directive info

  • This directive creates new scope.
