tm1-ui-subset-editor
directive in module tm1Ui

Description

This directive helps you to manage subsets of dimensions.

Directive Level APIs

refreshData()
For refreshing the list of data on this directive
getModelData()
For retrieving the data object of this directive.

Usage

as element:
<tm1-ui-subset-editor
       tm1-instance="{string}"
       tm1-dimension="{string}"
       tm1-hierarchy="{string}"
       tm1-subset="{string}"
       tm1-mdx="{string}"
       tm1-elements="{string}"
       [tm1-options="{object}"]
       [tm1-api="{object}"]
       [tm1-ui-class="{string}"]
       [ng-model="{object}"]>
</tm1-ui-subset-editor>

Directive info

  • This directive creates new scope.
