tm1-ui-dbra
directive in module tm1Ui

Description

A DBRA-like implementation for retrieving an element's attribute from TM1.

Directive Level APIs

onUpdate(status, data)
A function called after an update request has been initiated.
setContextTimeout(seconds)
A function, that sets how long will the context button be visible on activation. Default is 5 seconds.

Usage

as element:
<tm1-ui-dbra
       tm1-instance="{string}"
       tm1-dimension="{string}"
       tm1-element="{string}"
       tm1-attribute="{string}"
       [tm1-read-only="{boolean}"]
       [tm1-data-decimal="{string}"]
       [tm1-placeholder="{string}"]
       [tm1-validate="{string}"]
       [tm1-validate-message="{string}"]
       [tm1-hide-reference="{boolean}"]
       [tm1-api="{object}"]
       [tm1-on-load="{function}"]
       [tm1-on-change="{function}"]
       [tm1-change="{function}"]
       [tm1-ui-class="{string}"]
       [ng-model="{object}"]>
</tm1-ui-dbra>

Directive info

  • This directive creates new scope.

tm1-change

function
