tm1-ui-chart
directive in module tm1Ui

Description

Generate charts utilizing tm1-ui-chart-dbr as datasources.

Note that for 'lineWithFocus' chart types, there should be at least 2 data points at focus for the view finder to appear and to work correctly. This is as per the underlying chart's behaviour.

tm1-order-no will be auto-populated if it is not declared as an attribute. But if you need a specific order, this is still available for utilization.

Usage

as element:
<tm1-ui-chart
       tm1-chart-type="{line|lineWithFocus|discreteBar|multiBar|multiBarHorizontal|pie}"
       [tm1-height="{string}"]
       [tm1-show-values="{boolean}"]
       [tm1-data-decimal="{string}"]
       [tm1-format-percentage="{boolean}"]
       [tm1-margin="{object}"]
       [tm1-min-y="{string}"]
       [tm1-max-y="{string}"]
       [tm1-state="{string}"]
       [tm1-axis-label-x="{string}"]
       [tm1-axis-label-y="{string}"]
       [tm1-duration="{string}"]
       [tm1-hide-controls="{boolean}"]
       [tm1-hide-label="{boolean}"]
       [tm1-legend="{string|none}"]
       [tm1-ui-class="{string}"]
       [tm1-color-scheme="{Array}"]
       [tm1-rotate-label="{string}"]
       [tm1-options-override="{object}"]
       [tm1-accounting-settings-override="{object}"]
       [tm1-on-chart-element-select="{function}"]
       [ng-model="{object}"]>
</tm1-ui-chart>

A function that will be called after a chart element has been selected. The signature of the function that you can add should have 'chartElement' as one of the parameters.
