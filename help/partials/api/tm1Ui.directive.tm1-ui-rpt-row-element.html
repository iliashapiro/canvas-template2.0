tm1-ui-rpt-row-element
directive in module tm1Ui

Description

Use this to format and enable expanding/collapsing of your elements per row dimension. This directive is used in the TD section of your table element with the active form. Use as part of tm1-ui-rpt-view.

Usage

as attribute
<ANY tm1-ui-rpt-row-element="{object}"
     tm1-dimension="{string}"
     [tm1-freeze-element="{boolean}"]
     [tm1-ui-class="{string}"]>
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
tm1-ui-rpt-row-elementobject

The row property of the object defined on the parent tm1-ui-rpt-view. REQUIRED

tm1-dimensionstring

The dimension that will be referred to when expanding/collapsing. REQUIRED

tm1-freeze-element
(optional)
boolean

To disable collapsing/expanding on this element row

(default: "false")
