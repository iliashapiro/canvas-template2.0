tm1-ui-chart-dbr
directive in module tm1Ui

Description

A child directive to always use in conjunction with tm1-ui-chart as a data source.

For alternate hierarchy, consider a Product dimension used in a cube having a hierarchy by attribute named Location. To get the values of Bikes product, on the AU market:

tm1-elements=".., Bikes && Location::AU, .."

Usage

as element:
<tm1-ui-chart-dbr
       tm1-instance="{string}"
       tm1-cube="{string}"
       tm1-elements="{string}"
       [tm1-label="{string}"]
       [tm1-label-empty="{boolean}"]
       [tm1-series="{string}"]
       [tm1-area="{boolean}"]
       [tm1-order-no="{string}"]
       [tm1-color="{string}"]
       [tm1-show-movement="{boolean}"]>
</tm1-ui-chart-dbr>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails

tm1-area

boolean

To format a line chart as an area. Only need to be defined ONCE, and it will set the whole series as AREA, if used in a LINE chart.

(default: "false")

tm1-order-no
(optional)
string

The order or X-axis value of this data, if used in a LINE|WATERFALL chart.

(default: "NN")

tm1-color
(optional)
string

Assign a specific color to a data point in CSS color format (i.g. #FFXXXX or Red or rgb(1,1,1) ). Applicable to line charts only. For line series, the first color encountered on the DBR will be used on that whole series.

tm1-show-movement
(optional)
boolean

Used with WATERFALL chart. To indicate whether the bar rendered is movement only, or the full value.

(default: "true")
