tm1-ui-dbr-hidden
directive in module tm1Ui

Description

A DBR-like implementation for retrieving cube data from TM1 that is hidden and to be used with ng-model to retrieve numeric values

Usage

as element:
<tm1-ui-dbr-hidden
       tm1-instance="{string}"
       tm1-cube="{string}"
       tm1-elements="{string}"
       [tm1-refresh-group="{string}"]
       [tm1-sandbox="{string}"]
       [ng-model="{object}"]>
</tm1-ui-dbr-hidden>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
tm1-instancestring

Instance name for this directive to connect to. REQUIRED

tm1-cubestring

TM1 Cube name where this will connect to. REQUIREDWATCHED

tm1-elementsstring

comma separated element names that identifies the cube's data point to get. Element names can be an ALIAS. REQUIREDWATCHED

tm1-refresh-group
(optional)
string

An ID that will allow DBRs belonging to the same group to be refreshed only. Editing a DBR on this group refreshes only those having the same group as this.

tm1-sandbox
(optional)
string

Name of the sandbox that this DBR will be associated with.WATCHED

ng-model
(optional)
object

Angular JSON object where the actual values are stored, for usage in other parts of the page.
